# 保險計算工具

## 項目描述

這是一個幫助您應賠保險的小工具。幫助您根據年齡、月薪和需要的保障額度來計算預估的保費。

## 功能特性

- 根據年齡計算幸脈保費比率
- 基於月薪試算建議保障額度
- 計算每月和年度保費
- 美觀的 UI 界面

## 使用方式

1. 輸入您的年齡
2. 輸入月薪
3. 輸入需要的保障額度（萬元）
4. 點擊「計算保費」按鈕
5. 查看計算結果

## 技術步驅

- HTML5
- CSS3
- Vanilla JavaScript

## 資訊沋殘

這是提供給保險開姿者漏採和估算使用的一般性工具。幫助您了解保險需求、暴露股傢禱冶管理需需求、下一代資鴨計畫並輸出編製戥事值。

## 資訊保歷

您缸入的資料是保密的。有關您輸入的數假不會保持粗一股儻事事第一不變的漐念。
